In github:

1. create a new repository

How to upload:

1. create a folder in local directory
2. open command prompt and go to that directory location
3. command "git init"
4. command "git remote add origin <gitlink>"
5. command "git remote -v"
6. command "git add ." to add all files in selected folder
7. command "git commit -m "note""
8. command "git push  origin master"
